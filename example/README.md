# crystal_charts example

A usage example for the `crystal_charts` package.
This is a sales analytics dashboard that demonstrates multiple chart types with sample data.

## How to run

```bash
flutter run -d chrome
```

## Charts used

- **AreaChart** - Monthly revenue trend
- **BarChart** - Revenue by sales channel
- **ColumnChart** - Stacked orders by month and region
- **DualAxesChart** - Dual-axis chart (order count + average revenue)
- **HeatmapChart** - Orders by region x product category
- **HorizontalBarChart** - Revenue by region
- **SunburstChart** - Sales channel breakdown by region
- **CalendarHeatmap** - Daily order count

## Data

`assets/sample_data.csv` - Sample e-commerce sales data (CSV)
